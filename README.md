# Telegram Bot для генерации и проверки BIP39 сид-фраз

Бот генерирует валидные мнемонические фразы и проверяет балансы BTC, ETH, USDT (ERC20), BNB, USDT (BEP20).

## Развёртывание на Render

1. Загрузите все файлы (кроме `.env`) в репозиторий GitHub.
2. На [render.com](https://render.com) создайте Web Service, подключите репозиторий.
3. Добавьте переменные окружения:
   - `TELEGRAM_TOKEN` – токен от @BotFather
4. Build Command: `pip install -r requirements.txt`
5. Start Command: `python bot.py`

## Команды бота
- `/start` – приветствие
- `/generate` – сгенерировать новую фразу и показать балансы
- `/check <фраза>` – проверить балансы для указанной фразы
- `/stats` – статистика проверок

## Важно
Никогда не вводите реальные сид-фразы в непроверенные боты. Бот предназначен для образовательных целей.
